# Summary
This is an alfred workflow to take some text and converted it into a mocking
version. For example, the text "Donald Trump is a great president" would get
converted to "dOnAlD TrUmP Is a gReAt pReSiDeNt". This text is then copied to
the clipboard.

# How to use
Just type the keyword "mock" into alfred followed by the text that you want to
mock.
